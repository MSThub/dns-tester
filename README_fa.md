# DNS Checker

یک ابزار ساده ویندوزی برای تست چندین DNS سرور روی یک وب‌سایت هدف.

این ابزار به‌صورت خودکار:

* DNS سیستم را تغییر می‌دهد
* بررسی می‌کند که DNS به‌درستی اعمال شده باشد
* رزولوشن DNS را با `nslookup` تست می‌کند
* دسترسی‌پذیری یک URL هدف را بررسی می‌کند
* پینگ DNS سرورها را اندازه‌گیری می‌کند (در صورت امکان)
* بعد از اتمام، تنظیمات DNS را به DHCP برمی‌گرداند
* DNS سرورهای سالم را در یک فایل نتیجه ذخیره می‌کند

این ابزار چندین DNS سرور را روی یک URL هدف تست می‌کند.
در هر نتیجه، نام DNS، آدرس IP، وضعیت HTTP و زمان پاسخ (ping در صورت موفق بودن) نمایش داده می‌شود.
در پایان نیز یک خلاصه از نتایج و فایل خروجی ذخیره‌شده نمایش داده می‌شود.

## نیازمندی‌ها

* ویندوز
* Python 3.10+
* دسترسی Administrator

نصب وابستگی‌ها:

```bash
pip install requests
```

---

## فایل‌ها

| File               | Description                           |
| ------------------ | ------------------------------------- |
| `dns-checker.py`   | برنامه اصلی                           |
| `start.bat`        | اجرای اسکریپت با دسترسی Administrator |
| `dns_list.txt`     | لیست DNS سرورها                       |
| `dns_result_*.txt` | فایل نتیجه تولیدشده شامل DNSهای سالم  |

---

## فرمت لیست DNS

فایل `dns_list.txt` را ویرایش کنید و DNS سرورها را با این فرمت اضافه کنید:

```text
primary,secondary,name
```

نمونه‌ها:

```text
1.1.1.1,1.0.0.1,Cloudflare
8.8.8.8,8.8.4.4,Google
9.9.9.9,,Quad9
94.140.14.14
```

### نکات

* `secondary` اختیاری است
* `name` اختیاری است
* خطوط خالی نادیده گرفته می‌شوند
* خطوطی که با `#` شروع می‌شوند به‌عنوان کامنت در نظر گرفته می‌شوند

## نحوه استفاده

### گزینه 1 (پیشنهادی)

اجرا کنید:

```text
start.bat
```

فایل batch در صورت نیاز به‌صورت خودکار دسترسی Administrator درخواست می‌کند.

### گزینه 2

اسکریپت را به‌صورت دستی از داخل یک ترمینال Administrator اجرا کنید:

```bash
python dns-checker.py
```

## خروجی

بعد از پایان تست، همه DNS سرورهای سالم در یک فایل زمان‌دار ذخیره می‌شوند:

```text
dns_result_2026-06-23_15-30-22.txt
```

نمونه:

```text
primary: 1.1.1.1, secondary: 1.0.0.1, ping: 24ms
primary: 8.8.8.8, secondary: 8.8.4.4, ping: 31ms
```

## سلب مسئولیت

این ابزار در طول تست، تنظیمات DNS اینترفیس شبکه انتخاب‌شده را تغییر می‌دهد.

تنظیمات DNS بعد از اجرا به‌صورت خودکار به DHCP برگردانده می‌شوند، حتی اگر فرایند تست به‌طور ناگهانی متوقف شود.
